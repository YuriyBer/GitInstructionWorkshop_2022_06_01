### Установка git на ваш ПК

1. Windows
* Перейдите на страницу *https://git-scm.com/download/win*, и загрузка запустится автоматически.
2. Linux **Fedora, CentOS**
* $ sudo dnf install git-all
3. Linux **Debian, Ubuntu**
* sudo apt install git
4. Mac
* В версии Mavericks (10.9) и выше вы можете добиться этого просто первый раз выполнив 'git' в терминале.
* $ git --version
* Если Git не установлен, вам будет предложено его установить.
